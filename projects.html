    Android Based Projects

    These are some of my apps which are based on Android. This includes some of the apps that are requested by the users and one app is based on Machine Learning, while some of them are simple Android based Games. These games cover a wide range of programming concepts from simple Java programming to advanced Java features.

  1. Hostel Assist
    This is an Android Application that is developed to address the grievances of hostel students. Students can raise their complaints to their respective warden using this app. Wardens can address the complaints by taking appropriate action. It uses Google Firebase for its backend management.
  2. Heart Disease Detector
    This is an Android application that collects data from users and determines whether the user has Coronary Disease. This app employs a Machine Learning model which was trained to predict heart disease using an existing dataset. The model was deployed to an Android-based app using a TensorFlow Keras Model.
  3. Age and Gender Predictor
    This is an Android application that predicts the Age group of a user and their Gender based on user image. This app employs a Deep Learning model which was trained to predict Age and Gender using Facial Recognition. The model was deployed to an Android-based app using a TensorFlow Keras Model.
  4. GoT Character Predictor
    This is an Android application that can be used to predict the character from Game of Thrones web series. It accepts an input image from user and predicts which character it is. This app employs a Machine Learning model which was trained to predict GoT character using an existing image dataset. The model was deployed to an Android-based app using a TensorFlow Keras Model.
  5. GoT Akinator
    This is an Android application that predicts the character from GoT which you are thinking. Think of it as an Akinator specifically for Game of Thrones characters. It asks set of questions and based on response provided by user, it predicts the character. It follows Tree Traversal concept for prediction.
  6. Hangman Game

  7. Guess the Song - Game
